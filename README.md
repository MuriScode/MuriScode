## 🚀 Sobre mim

Atualmente curso Ciência da Computação na UFABC e atuo como estagiário no NETEL/UFABC, onde trabalho com manutenção de sistemas, desenvolvimento de dashboards e estruturação de dados.

Gosto de atuar na intersecção entre o desenvolvimento front-end e a engenharia de dados. Meu foco principal é transformar problemas complexos em soluções digitais que gerem impacto real, seja criando **Design Systems** para melhorar a experiência do usuário ou estruturando **APIs e Dashboards** para suporte à tomada de decisão.

Sempre aberto a novos desafios que envolvam tecnologia, arte, dados e colaboração! 💡🎶
