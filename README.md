## 🚀 Sobre mim

Atualmente curso Ciência da Computação na UFABC e atuo como estagiário no NETEL/UFABC, onde trabalho com manutenção de sistemas, desenvolvimento de dashboards e estruturação de dados.

Gosto de atuar na intersecção entre o desenvolvimento front-end e a engenharia de dados. Meu foco principal é transformar problemas complexos em soluções digitais que gerem impacto real, seja criando **Design Systems** para melhorar a experiência do usuário ou estruturando **APIs e Dashboards** para suporte à tomada de decisão.

Vivo na intersecção entre arte e tecnologia, é onde as soluções mais interessantes aparecem pra mim. 🎶

## 🌟 Projetos em destaque

### 📉 [Educational Dropout Analysis](https://github.com/MuriScode/educational-dropout-analysis)
Projeto de análise de dados educacionais desenvolvido com foco na identificação de padrões de evasão no curso de Especialização em Geoprocessamento (ofertas 2023 e 2024).
> **Tecnologias:** Python (Pandas), Power BI, Google Colab, Excel

### 🧩 [QZero/UAB Components](https://github.com/MuriScode/q-zero-mini-design)
Biblioteca de componentes UI (Mini Design System) desenvolvida para padronização de interfaces educacionais na plataforma Moodle.
> **Tecnologias:** HTML5, CSS3 avançado (Scroll Snap, Grid/Flexbox), Foco em UX/UI e Acessibilidade.

### 🎨 ArtGuideSP (Acadêmico)
Plataforma web colaborativa para mapeamento cultural geolocalizado na cidade de São Paulo.  
> **Tecnologias:** Node.js, JavaScript, Leaflet.js, modelagem estrutural em JSON.  
> *Nota: O código-fonte original está privado por diretrizes acadêmicas, mas estou sempre à disposição para bater um papo sobre a arquitetura do projeto!*
